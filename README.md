# Student Task Manager API with MongoDB

## Project Overview

Student Task Manager API is a backend application developed using Node.js, Express.js, MongoDB and Mongoose. This project demonstrates database integration, CRUD operations, API development and data management techniques. It was developed as part of the DecodeLabs Full Stack Development Internship – Project 3.

The application allows users to create, retrieve, update and delete tasks while storing data permanently in a MongoDB database.

---

## Project Objectives

* Design and implement a database schema
* Integrate MongoDB with a Node.js application
* Perform CRUD (Create, Read, Update, Delete) operations
* Implement API routing and request handling
* Ensure proper data validation and management
* Understand backend-to-database communication

## Features

### Create Task

Create and store new tasks in the database.

### Retrieve Tasks

Fetch all tasks or a specific task using its unique ID.

### Update Task

Modify existing task details and status.

### Delete Task

Remove tasks from the database permanently.

### Database Integration

Store and retrieve data using MongoDB and Mongoose.

### Data Validation

Validate user input before database operations.
